# ğŸ“– [Vet clinic database] <a name="about-project"></a>

**[Vet clinic database]** is a relational database to create the data structure for a vet clinic which creates a complete database with data about:
- **[animals]**
- **[animals owners]**
- **[clinic employees]**
- **[visits]**
